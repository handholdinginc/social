About Handholding Social

Site rules

Please refrain from:

  1. Posting anything illegal.
  2. Intentionally creating multiple accounts to circumvent people blocking you.
  3. Being a bot (unless you post unlisted and do not auto-follow).
  4. Any account setup solely to advertise a product or service.

Free And Open Source Software

Handholding Social is free and open source (FOSS) software based on Soapbox which runs atop a Pleroma server
